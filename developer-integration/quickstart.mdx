## Overview

The Serpentic API allows developers to build dashboards, bots, leaderboards, analytics tools, and more on top of the game.

**Base URL:** `https://api.serpentic.lol`

All endpoints are **public** (no authentication required yet) with rate limiting applied at 60 requests per minute per IP.

## Quick Example

```typescript
const response = await fetch('https://api.serpentic.lol/health');
const data = await response.json();

console.log(data);
```

<Info>
All responses follow a consistent structure:

```json
{
  "success": true,
  "data": { },
  "timestamp": "2026-05-20T13:45:12Z"
}
```
</Info>

## Available Endpoints

| Endpoint | Method | Description |
|---|---|---|
| `/health` | `GET` | API health and server status |
| `/player/{wallet}` | `GET` | Player statistics and history |
| `/global/stats` | `GET` | Network-wide statistics |
| `/leaderboard` | `GET` | Global leaderboard |
